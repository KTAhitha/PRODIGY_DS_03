# PRODIGY_DS_03

## Data Science Internship - Task 03

This project is part of my Data Science Internship at Prodigy Infotech.

### Objective
Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on their demographic and behavioral data.

### Dataset
Bank Marketing Dataset

### Tools & Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

### Steps Performed
- Loaded and explored the dataset
- Converted categorical data into numerical format
- Split the data into training and testing sets
- Built a Decision Tree Classifier model
- Evaluated the model using accuracy score
- Visualized the decision tree

### Output
- Prediction of customer purchase (Yes/No)
- Accuracy score of the model
- Visualization of Decision Tree

### Key Insights
- Decision trees help in understanding how decisions are made step-by-step
- Features like job, age, and balance influence customer decisions
- The model can predict customer behavior with good accuracy
